Paso 4: Finalizar la Instalación

Pasos para la instalación de BIZUIT 4.0 ›› Instalación de BIZUIT Server ››

Si se realizaron instalaciones previas de BIZUIT Server, puede haber quedado registrado un certificado de seguridad que permite la comunicación segura entre el servidor de BIZUIT Server y sus clientes. En ese caso aparecerá la siguiente pantalla en la cual deberá seleccionar la opción NO para evitar tener que re-distribuir manualmente este certificado en sus clientes.




Si el nombre de la base de datos selecionada en la Configuración Básica de Server Database ya existe, se le presentará la siguiente pantalla solicitando su confirmación, tenga en cuenta que al confirmar esta operación se sobrescribirán los datos de dicha base de datos.



Durante el proceso de instalación pueden aparecer ventanas de DOS como la siguiente, simplemente espere que termine el proceso y se cierre la ventana:



Si en la pantalla de Licenciamiento de BIZUIT Server selecciónó la opción “Register license On-Line” se mostrará la siguiente ventana:




Si en la pantalla de Licenciamiento de BIZUIT Server seleccionó la opción “Community Edition” el programa de instalación mostrará la siguiente ventana.



Tenga en cuenta mientras no registre una licencia según lo indicado en la sección "Herramienta de Licenciamiento” podrá ejecutar el software normalmente pero con las limitaciones de la versión "Community Edition"
