Paso 1: Iniciar Asistente de Instalación

Pasos para la instalación de BIZUIT 4.0 ›› Instalación de BIZUIT Dashboard Server ››

Para comenzar la instalación se debe abrir una ventana de comando en modo Administrador:


Si el lenguaje de su sistema operativo es inglés, al realizar una busqueda de cmd en el panel de busquedas de Windows, seleccionar "Command Prompt", hacer click con el botón secundario y seleccionar la opción "Run as Administrator".

Si el lenguaje de su sistema operativo es español, al realizar una busqueda de cmd en el panel de busquedas de Windows, seleccionar "Símbolo del sistema", hacer click con el botón secundario y seleccionar la opción "Ejecutar como Administrador".



En la ventana de comando, posicionarse en el directorio donde se encuentre el archivo "Tycon BIZUIT Dashboard 4.0.0.msi" y ejecutarlo:
